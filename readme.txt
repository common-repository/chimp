Migrate your WordPress data to Chimp. Posts, comments, users & media.

== Description ==

The Chimp WordPress plugin allows you to export your WordPress posts, comments, users & media files to [Chimp](https://chimphq.de).

This plugin is currently available in german language only.


== Installation ==

The automatic plugin installer should work for most people. Manual installation is easy and takes fewer than five minutes.

1. Create a `chimp` directory in your `plugins` directory. Typically that's `wp-content/plugins/chimp/`.
2. Into this new directory upload the plugin files (`chimp.php`, etc.)
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to Settings > Chimp and copy your API URL and API Key
5. In Chimp go to Settings > Data > Import (Einstellungen > Daten > Import ) and create a new import with your API URL and API Key
